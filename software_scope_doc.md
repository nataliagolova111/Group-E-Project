**1. Project Overview:**

This Inventory Manager program allows the user to add items and record their corresponding quantities. The user can view all stored inventory and update quantities when changes occur. It helps small businesses keep track of their inventory. The program is menu-driven, which allows the user to choose between different actions. All inventory data is saved to a text file, which allows the data to be saved even after the program is run again. 

 

**2. Target User:**

The target user is a small business owner who needs a simple way to track inventory items and their quantities. The program is designed for a single user and is intended to run in the VS Code Terminal. It is suitable for users with very basic computer knowledge who prefer a text-based program. 

 

**3. Functional Requirements (Must-have Features):**

- Add a new inventory item with a name 

- Record a quantity for each item when it is added 

- View all inventory items with their corresponding quantities 

- Update the quantity of an existing item 

- Display inventory items with their quantities in an organized way 

- Extension: Menu-driven user input 

- Save all inventory data to a text file 

- Load existing inventory data when the program starts 

 

**4. Non-Functional Requirements (Constraints):**

- Written in Python 

- Single-user program 

- Text-based program, no graphical interface 

- Runs in the VS Code Terminal 

- Organized into multiple Python files (separate functions) 

 

**5. Out-of-Scope Features:**

- Graphical user interface 

- Mobile application 

- Login systems 

- Use of classes or object-oriented programming 

- Sorting algorithm
