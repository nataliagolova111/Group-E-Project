# Inventory Manager

## Description

This program is a simple Inventory Manager made by Python. It allows user to add items, set quantities and view all items, and update quantities.All data is saved in a text file which allows them be used again next time. This program can be used in small businesses like shops to keep track of inventory.

## How to Run

1. Open Group-E-Project/Main code/Python Code folder in VS Code 
2. Open demo.py file
3. Run demo.py file
4. Interact with the menu (use numbers 1-4 to navigate)

## Example input/output

- Input: 1 → Apple → 50
- Output: Saved: Apple | Quantity: 10

- Input: 2
- Output: 
Inventory List:

1 - Name: Apple | Quantity: 50

- Input: 3 → Apple → 100
- Output: Quantity updated successfully!

- Input: 5
- Output: Invalid choice. Try again.
